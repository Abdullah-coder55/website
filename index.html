ESP32 Projects

ESP32 CAM Surveillance

This project transforms an ESP32-CAM module into a low-cost IP surveillance camera. It streams live video through a web interface and integrates servo motors to provide pan and tilt movement, allowing full directional control remotely through the same browser-based dashboard.

Wireless Car Control

Uses an ESP32 to wirelessly control an Arduino-based robotic car over Bluetooth. ESP32 sends button commands, and Arduino interprets them to move the car forward, backward, left, or right using an L298N motor driver. The setup enables wireless driving without Wi-Fi.
