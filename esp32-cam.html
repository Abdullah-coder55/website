Description

This project turns the ESP32-CAM module into a smart surveillance camera capable of live streaming through a web interface. With attached servo motors, the user can control camera direction (pan/tilt) remotely from the same webpage.

Components Needed

  • ESP32-CAM Module
  • Servo Motor (x2)
  • FTDI Programmer
  • Jumper Wires
  • Power Supply (5V)
