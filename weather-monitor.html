Description

This project displays real-time temperature and humidity readings on an OLED screen using an Arduino Uno and DHT11 sensor. It's ideal for indoor environmental monitoring and educational use.

Components Needed

  • Arduino Uno
  • DHT11 Temperature and Humidity Sensor
  • 0.96" OLED Display (I2C)
  • Breadboard + Jumper Wires
  • USB Cable / Power Supply
