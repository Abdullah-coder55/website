Description

This project uses an Arduino Uno, a touch sensor, a speaker, and an OLED display to create a basic security system. The system monitors whether an object (e.g., a valuable item) is present on the sensor or not.

Working Principle

When an object is in contact with the touch sensor, the system remains in a safe/standby mode. If the object is removed or the sensor is no longer touched, the system immediately:

Displays a warning message on the OLED screen Activates a buzzer or speaker to generate an audible alarm

Components Required

  • Arduino Uno
  • Touch Sensor
  • Wires and Breadboard/li>
  • small speaker
  • 0.96" I2C OLED Display
