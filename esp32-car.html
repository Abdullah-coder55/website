Description

This project enables wireless control of a robotic car using the ESP32 to send Bluetooth signals to an Arduino Uno, which interprets the signals to control movement via an L298N motor driver. The car responds to directional commands such as forward, backward, left, and right.

Components Needed

  • ESP32 Dev Board
  • Arduino Uno
  • L298N Motor Driver
  • DC Motors
  • HC-05/HC-06 Bluetooth Module
  • Power Source
  • Chassis & Wheels
  • Jumper Wires
