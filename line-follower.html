Description

This robot is designed to follow a black line on a white surface using IR sensors and an Arduino Uno. It adjusts the motor speeds depending on sensor readings to stay on track, making it useful for automation and line-based navigation tasks.

Components Needed

  • Arduino Uno
  • IR Sensor Module x2
  • L298N Motor Driver
  • DC Motors + Wheels
  • Chassis
  • Jumper Wires
  • Battery Pack
